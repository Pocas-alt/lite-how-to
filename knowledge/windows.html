Windows klavišų kombinacijos

Naudingos „Windows“ spartieji klavišai kasdieniam darbui:

  • Ctrl + C: Kopijuoti
  • Ctrl + X: Iškirpti
  • Ctrl + V: Įklijuoti
  • Ctrl + Z: Atšaukti veiksmą
  • Ctrl + A: Pažymėti viską
  • Ctrl + S: Išsaugoti
  • Ctrl + O: Atidaryti
  • Ctrl + P: Spausdinti
  • Ctrl + F: Paieška dokumente
  • Ctrl + H: Pakeisti (Find & Replace)
  • Alt + Tab: Perjungti tarp atidarytų programų
  • Alt + F4: Uždaryti aktyvią programą/langą
  • Ctrl + Shift + Esc: Atidaryti Užduočių tvarkytuvę
  • Ctrl + Alt + Delete: Saugos meniu (užrakinimas, atsijungimas, Užduočių tvarkytuvė ir kt.)
  • Win + D: Rodyti/slėpti darbalaukį
  • Win + E: Atidaryti Failų naršyklę
  • Win + L: Užrakinti kompiuterį
  • Win + V: Iškarpinės istorija (įjungus Windows nustatymuose)
  • Win + Shift + S: Ekrano iškarpos įrankis (snipping)
  • Win + R: „Run“ (vykdyti) dialogas
  • Win + S: Paieška
  • Win + . (taškas): Emoji ir simbolių skydelis
  • Win + Rodyklės: Langų išdėstymas (snap) į kairę/dešinę/viršų/apačią
  • Win + Ctrl + D: Naujas virtualus darbalaukis
  • Win + Ctrl + ←/→: Perjungti virtualius darbalaukius
  • PrtScn arba Win + PrtScn: Ekrano nuotrauka (Win+PrtScn išsaugo automatiškai)
  • F2: Pervardyti pasirinktą failą
  • F5: Atnaujinti
